## üá¨üáß English

This project is an **AI-based Tomato Detection System** that classifies tomatoes as *ripe* or *unripe* using a deep learning model trained on image data.

### üß† Model
The model (`tomate_detection_weight.pth`) is available on [Hugging Face](https://huggingface.co/myseringan1/tomato-detection-model).  
It is built with **PyTorch**, and uses **transfer learning** for efficient and accurate detection.

### üöÄ Features
- Real-time tomato detection
- Supports webcam input
- Trained with dataset of ripe/unripe tomatoes
- Easy to integrate with embedded systems (e.g. ESP32-CAM)
